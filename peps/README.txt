Pour pouvoir utiliser le projet, il faut installer une base de données.
Et modifier les chaines de connexions qui se trouvent dans les fichiers :
	- App.Config du projet DALAssetDotNet
	- Web.config dans le projet PepsClient
Il est possible de visualiser les derniers résultats obtenus sans avoir à configurer la base de données
en lançant le projet GrapTest.
Un dump de notre base de données est disponible à la racine du projet sous le nom peps.bak
